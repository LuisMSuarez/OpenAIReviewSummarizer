# OpenAI Review Summarizer
Site that showcases automated creation of product review summaries using OpenAI or open source models (HuggingFace)
Lightweight web app displays product reviews (from a MySql database) and produces concise, actionable summaries using OpenAI models. Built with React, TypeScript and modern web tooling, the Review Summarizer helps teams quickly understand sentiment, common issues, and highlights across large sets of reviews.
